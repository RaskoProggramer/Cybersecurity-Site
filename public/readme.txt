README - Security Policies Package
 
This package contains essential security policy documents and resources:
 
1. acceptable_use_policy.pdf
   - Clear rules for responsible use of IT resources.
 
2. password_policy.pdf
   - Modern password/passphrase requirements with MFA coverage.
 
3. incident_response_plan.pdf
   - Template including roles, phases, and a starter runbook.
 
4. phishing_defense_guide.pdf
   - Best practices guide covering people, process, and technology controls.
 
5. phishing_cheat_sheet.png
   - Infographic with quick reference tips for phishing defense.
 
Use these documents to strengthen organizational cybersecurity awareness and resilience.
